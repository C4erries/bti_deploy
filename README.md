- Клиент: `client@example.com` / `client123`

## 3D demo

После первого запуска (или пересоздания SQLite) сиды создают один тестовый заказ для клиента с готовым 2D+3D планом.

Как проверить 3D:

1. Подними проект:
   ```bash
   docker-compose up --build
   ```
2. Зайди на фронтенд: http://localhost:5173
3. Залогинься как клиент:
   - email: `client@example.com`
   - password: `client123`
4. Открой раздел клиента и список заказов, выбери тестовый заказ (обычно первый).
5. В деталях заказа переключись на 3D (или 2D/3D‑переключатель).
6. На 3D‑вьюере:
   - увидеть готовый 2D план и пример 3D‑объекта (sofa),
   - подвигать/повертеть сцену,
   - любые изменения 3D‑объектов сохраняются в `plan.objects3d` через уже существующий `POST /api/v1/client/orders/{orderId}/plan/changes`.
